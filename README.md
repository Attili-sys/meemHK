# meemHK Landing Page

Premium, high-converting static landing page for meemHK on noon Saudi.

## Features
- Fully static (HTML, CSS, JS) - no build steps, no backend
- Mobile-first responsive design
- RTL/LTR support with Arabic (default) and English toggle
- Smooth scroll animations via Intersection Observer
- Modern CSS (variables, grid, clamp typography)
- Ready for GitHub Pages deployment

## Deployment Instructions (GitHub Pages)
This repository is configured to be a static website. To deploy on GitHub Pages:

1. Push this code to a GitHub repository.
2. Navigate to your repository **Settings** on GitHub.
3. Click on **Pages** in the left sidebar.
4. Under "Build and deployment", set the **Source** to `Deploy from a branch`.
5. Under "Branch", select your main branch (e.g., `main` or `master`) and folder `/(root)`.
6. Click **Save**.
7. GitHub will start building the site. Within a few minutes, your site will be live at `https://<your-username>.github.io/<your-repo-name>/`.

## Tech Stack
- **HTML5**: Semantic and accessible.
- **CSS3**: Variables, Grid, Flexbox, prefers-reduced-motion support.
- **Vanilla JavaScript**: Language switching, dynamic footer year, simple click tracking placeholder, fade-in animations.
